# Submissions Ledger

> Copy this to `submissions.md` at the project root. **Single source of truth** for every claim.
> One row per claim. Update `Status` and `Date paid` as claims progress.

| Date filed | Vendor / Purpose | Folder | Period | Amount (orig) | Amount (home) | Status | Date paid | Notes |
|---|---|---|---|---|---|---|---|---|
| 2026-04-15 | [Example vendor] | `software_subscriptions/2026_example/` | Apr 2026 | USD 200.00 | EUR 184.61 | Filed | ? | — |
| | | | | | | Drafting | | |

**Status values:** `Drafting` (form being prepared) · `Filed` (printed, signed, handed off) · `Paid` (reimbursed) · `Rejected` · `Historical` (pre-ledger).

**Discipline:** a row exists for *every* submission folder. A `Filed` row with no `Date paid` after 60 days is *stale* — chase the admin contact.
